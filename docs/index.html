Introduction

Introduction

This is a gitbook project for the documenation of the Arachne Bot Framework.

This static website is deployed on the following dokku instance: http://arachne-docs.intech-lab.com/.

To build locally the documentation, or build the documentation for another version, follow these instructions:

  1. Fork, clone or download this project.
  2. [Install][] GitBook npm install gitbook-cli -g
  3. Fetch GitBook's latest stable version gitbook fetch latest
  4. Build the documentation website: gitbook build
  5. Double click the index.html file in the _book folder (or run an http-server it the same folder).
